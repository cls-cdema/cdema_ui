[react-dropzone](https://github.com/react-dropzone/react-dropzone) allows using simple react hooks to create a HTML5-compliant drag'n'drop zone for files.

### css classes

The following css classes are available to apply the UI-Kits styling to `react-dropzone`:

| Branch                               | Build Status                                                             |
| ------------------------------------ | ------------------------------------------------------------------------ |
| .wfp--dropzone                       | Overall drag-and-drop area                                               |
| .wfp--dropzone\_\_input--drag-active | Active class from the drag-and-drop area if a drag is currrently ongoing |
| .wfp--dropzone\_\_icon               | Upload icon shown inside the drag-and-drop area                          |
| .wfp--dropzone\_\_file-list          | File list showing currently quewed files                                 |

### Example

This example shows the advanced usage featuring a drop-down area and a list of files.

```jsx
import Dropzone, { useDropzone } from 'react-dropzone';
import { Icon, List, ListItem } from '@wfp/ui';
import { iconUpload } from '@wfp/icons';
import classNames from 'classnames';

export const Advanced = (args) => {
  const {
    acceptedFiles,
    getRootProps,
    getInputProps,
    isDragActive,
  } = useDropzone();

  const files = acceptedFiles.map((file) => (
    <ListItem key={file.path}>
      {file.path} - {Math.round(file.size / 1000)} kB
    </ListItem>
  ));

  const className = classNames('wfp--dropzone__input', {
    'wfp--dropzone__input--drag-active': isDragActive,
    [className]: className,
  });

  return (
    <section className="wfp--dropzone">
      <div {...getRootProps({ isDragActive, className: className })}>
        <input {...getInputProps()} />
        <Icon className="wfp--dropzone__icon" icon={iconUpload} />
        <div>Drop files or click here to upload</div>
      </div>
      <aside className="wfp--dropzone__file-list">
        <h4>Files</h4>
        <List>{files}</List>
      </aside>
    </section>
  );
};
```
